## 📖 About

**Smart Housing Connect** is a modern housing society management platform that brings residents, complaints, marketplace, and community activities together in one application.

Developed as an independent portfolio project under **✨ Swapno Labs ✨**.

---

## ✨ Features

### 🏠 Home
- Modern Landing Page
- Glassmorphism UI
- Responsive Design

### 📋 Complaint Module
- Raise Complaint
